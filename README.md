# Montenegro Route Map

Полная версия сайта для GitHub Pages.

## Структура

- `index.html` — общая карта маршрута;
- `cities/` — отдельные страницы городов;
- `bookings/` — PDF-подтверждения бронирований.

На главной странице в блоке «Карты городов» у каждого города есть две кнопки:
- «Карта города»;
- «Google Maps».

На страницах отдельных городов также есть кнопка «Открыть маршрут в Google Maps».
